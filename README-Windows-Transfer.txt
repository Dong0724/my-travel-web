My Travel Web - Windows 10 / Windows 11 transfer note

How to move:
1. Unzip the package to any folder on the new Windows computer.
2. Double-click start-server.bat.
3. Open http://127.0.0.1:4173/ if the browser does not open automatically.

Runtime requirement:
- Preferred: Python 3
- Fallback: Node.js with npx

Notes:
- This is a static frontend project.
- Weather data is fetched from Open-Meteo in the browser.
- YouTube live camera links open YouTube search pages.
- Keep the assets folder next to index.html, app.js, and styles.css.
